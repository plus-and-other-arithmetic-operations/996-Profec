## Setup

Install the required fonts

Add the script entry into the car's ext_config

```ini
[SCRIPTABLE_DISPLAY_...]
ACTIVE = 1
RESOLUTION = 1024, 1024
DISPLAY_POS = 0,0  
DISPLAY_SIZE = 1024, 1024
SKIP_FRAMES = 0
MESHES = PROFEC_MONITOR
SCRIPT = profec.lua
```
